## 📊 CAMPAIGN STRUCTURE

### Phase 1: Testing (Week 1-2)
**Budget:** €50/day
**Structure:**
- 1 Campaign (Conversions/Leads)
- 3-4 Ad Sets (different audiences)
- 5-6 Ads per ad set (mix of copy + creative)
- Broad targeting to let algorithm learn

### Phase 2: Scaling (Week 3+)
**Budget:** Scale winners to €100-200/day
- Kill ads under 1% CTR
- Scale ads with CPA under €40
- Create lookalikes from converters
- Test new hooks on winning audiences

### Testing Priority:
1. Test HOOKS first (keep creative same)
2. Test AUDIENCES second (keep winning ad)
3. Test CREATIVES third (keep winning hook + audience)
4. Test OFFERS last (if CPA still too high)
