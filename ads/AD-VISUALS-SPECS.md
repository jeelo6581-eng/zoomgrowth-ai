## ✅ CREATIVE CHECKLIST

Before launching any ad:

- [ ] Text is readable on mobile
- [ ] Brand colors consistent
- [ ] Logo visible but not dominant
- [ ] Clear CTA button/text
- [ ] Value prop in first 3 seconds (video)
- [ ] Hook in first line (copy)
- [ ] Landing page matches ad promise
- [ ] UTM parameters added
- [ ] Multiple variations for testing

---

## 🧪 A/B TEST PRIORITY

**Test in this order:**

1. **Hooks** — Test 5 different first lines, same creative
2. **Visual style** — Dark vs light, photo vs graphic
3. **CTA** — "Book Call" vs "Learn More" vs "Get Started"
4. **Social proof** — With stats vs without
5. **Length** — Short punchy vs detailed story
